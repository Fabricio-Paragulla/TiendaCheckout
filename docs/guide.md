# Guía de Usuario

## Cómo completar una compra

### Paso 1: Datos de Facturación
Introduce tus datos personales.
* **Validación inteligente:** Al introducir un CP válido (ej: `28001`, `08001`), la ciudad y provincia se rellenan solas.
* **Nota:** El NIF debe tener formato válido (8 números y letra).

### Paso 2: Dirección de Envío
Puedes usar la misma dirección de facturación marcando la casilla.
* Si desmarcas la casilla, deberás rellenar los datos del destinatario.

### Paso 3: Método de Pago
Selecciona tu método preferido:
* **Tarjeta:** Se valida automáticamente el número (algoritmo Luhn) y detecta si es Visa/Mastercard.
* **PayPal / Bizum:** Formularios simplificados.
* **Cupón:** Prueba a introducir el código `VERANO20` para un 20% de descuento.

### Paso 4: Confirmación
Revisa el resumen de tu pedido. Debes aceptar los **Términos y Condiciones** y la **Política de Privacidad** para habilitar el botón de confirmar.
