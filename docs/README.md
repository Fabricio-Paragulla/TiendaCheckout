## Introducción

Este proyecto implementa un sistema de checkout completo siguiendo las especificaciones de la **Práctica 5.1**. Utiliza tecnologías modernas como Vue 3, TypeScript, Vite y Vitest.

### Requisitos cumplidos
- ✅ Wizard de 4 pasos
- ✅ Validación de DNI, Tarjetas (Luhn) y CP (Asíncrono)
- ✅ Tests con cobertura >80%
- ✅ Documentación con VuePress
